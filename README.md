# 📜 ЛЕТОПИСЬ ПРОЕКТА: БАЗА СТУДЕНТОВ (ВЕ-КРЕСТ-КРЕСТ)

## 🏰 О сём труде

Сей код есть программа для ведения **летописи студентов**, писанная на языке C++,
но облачённая в древнеславянские речения и макросную магию, именуемую
`Ве_крест_крест.h`.

Программа дозволяет хранить, искать, изменять и упорядочивать записи о студиозусах,
а также сберегать их в файл летописи `students.txt`.

---

## ⚙️ Возможности

В сем труде возможно:

- 📖 Показати всех студентов из летописи
- ➕ Добавити нового студента
- 🔍 Отыскати студента по номеру
- ✏️ Правити запись студента
- ❌ Удалити студента из летописи
- 🔢 Сортировать студентов:
  - по имени
  - по группе
  - по среднему баллу
  - по году поступления
- 💾 Сберечь летопись в файл
- 📥 Загрузить летопись при запуске

---

## 🗂️ Хранение данных

Данные студентов пишутся в файл:

```

students.txt

```

Формат строки в летописи:

```

номер|имя|группа|средний_балл|год_поступления

```

---

## 🧑‍🎓 Структура студента

Каждый студент есть запись вида:

- номер (целина)
- имя (много_букав)
- группа (много_букав)
- средний балл (вель_дробь)
- год поступления (целина)

---

## 🧭 Управление (меню программы)

При запуске предстаёт меню:

```

1. Показать всех студентов
2. Добавить студента
3. Найти студента по номеру
4. Править студента
5. Удалить студента
6. Сортировать студентов
7. Сберечь летопись
8. Выйти

````

---

## 🏗️ Сборка

Собирати сей проект надлежит компилятором C++ (желательно C++17 или новее):

```bash
g++ VediYchanikov.cpp -o students
````

---

## 🧙 Особенности языка

Сей проект использует заголовок:

```
Ве_крест_крест.h
```

в коем скрыты макросы, заменяющие привычные слова C++ на древнерусские аналоги:

* `молвить` → cout
* `внемлить` → cin
* `забор` → vector
* `летопись` → file stream
* `путевой_камень` → switch
* `путь` → case
* `коли` → if

и многое иное, дабы код выглядел как летопись древняя.
