# Бꙋкы Стꙋдентьскыя

**База данных студентов** — сiя малая летопись сотворена на языке **ВЕ✟✟** и служитъ для храненiя, поиска и правки сведѣнiй о студентахъ.

## О дѣлѣ семъ

Программа являетъ собою консольную летопись, гдѣ можно вписывать студентовъ, зрѣть весь списокъ, искать по номеру, править запись, удалять лишнее и упорядочивать списокъ по разнымъ признакамъ.

Данныя сберегаются въ файлѣ `students.txt`, дабы послѣ нового запуска память о студентахъ не пропадала.

## Что умѣетъ летопись

- Показати всѣхъ студентовъ.
- Добавити нового студента.
- Найти студента по его номеру.
- Правити имя, группу, среднiй баллъ и годъ поступленiя.
- Удалити запись изъ летописи.
- Сортировати студентовъ:
  - по имени;
  - по группѣ;
  - по среднему баллу;
  - по году поступленiя.
- Сберегати записи въ `students.txt`.
- Загружати прежнiя записи при новомъ запускѣ.

## Какъ пустити программу

Открой проектъ `VediYchanikov.sln` въ средѣ сборки и запусти его.

Послѣ запуска явится меню:

```text
=== База данных студентов ===
1. Показать всех студентов
2. Добавить студента
3. Найти студента по номеру
4. Править студента
5. Удалить студента
6. Сортировать студентов
7. Сберечь летопись
0. Выйти
```

Введи число пути и слѣдуй указанiямъ на экранѣ.

## Какiя поля хранитъ студентъ

Каждая запись имѣетъ:

- номеръ;
- имя и фамилiю;
- группу;
- среднiй баллъ;
- годъ поступленiя.

Номеръ даётся программой самъ, по порядку.

## О храненiи

При выходѣ чрезъ путь `0` летопись сберегается въ файлъ `students.txt`.

Строки въ файлѣ хранятся въ простомъ видѣ:

```text
номер|имя|группа|средний_балл|год
```

Файлъ можно не творити вручную: коли его нѣтъ, программа начнётъ съ пустой летописи.

## О языке

Сiя работа написана на **ВЕ✟✟** — древне-русскомъ нарѣчiи для творенiя программъ, гдѣ обычныя повелѣнiя замѣнены словесами родной старины:

```text
молвить       — вывести слово
внемлить      — принять вводъ
коли          — условiе
покуда        — повторенiе
забор         — собрание записей
летопись      — файлъ памяти
```

Да будетъ летопись крепка, а студенты сысканы безъ труда.
